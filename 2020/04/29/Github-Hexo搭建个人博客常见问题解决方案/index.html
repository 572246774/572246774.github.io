常见问题解决方案：

1、未找到相关得Issues进行评论

解决方案：点击蓝色按钮，使用GitHub登录，登录成功即可。如果登录成功后直接跳转到GitHub中，则需要检查GitHub设置，如下：

​ Settings > Developer setting > OAuth Apps

注意上面图片中的callback URL即可，如果有个人域名就填写个人域名地址，没有填homepage URL
